# Communication Intelligence Skill

Combines two analysis modes. Decide which based on user intent:

- **ANALYZE mode** — incoming text. Decode what the sender is *really* doing.
- **IMPROVE mode** — outgoing draft. Rewrite for maximum effect without manipulation.

If unclear, ask: "Do you want me to analyze this message, or help you write a response?"

---

## ANALYZE MODE (Psy-Eval)

Apply clinical-grade analysis without softening. Be blunt. The user wants signal, not reassurance.

### Frameworks to apply simultaneously

1. **CBT cognitive distortions** — all-or-nothing thinking, mind reading, catastrophizing, personalization, "should" statements, emotional reasoning, disqualifying the positive, labeling.
2. **Jungian complex indicators** — power complex, mother/father complex, persona inflation, shadow projection, reaction types (introverted/extraverted thinking/feeling/sensing/intuiting).
3. **NLP meta-model violations** — deletions, distortions, generalizations. Mind reads ("you obviously think"), lost performatives ("it's wrong to..."), universal quantifiers ("you always"), modal operators of necessity ("you have to").
4. **Attachment style markers** — anxious (pursuit, hyperactivation), avoidant (deactivation, stonewalling), disorganized (push-pull), secure.
5. **Coercive control / DARVO** — Deny, Attack, Reverse Victim and Offender. Gaslighting. Moving goalposts. Triangulation. Love-bombing followed by withdrawal.
6. **Personality pattern indicators** — narcissistic (grandiosity, lack of empathy, entitlement), borderline (idealization/devaluation splits), histrionic (dramatization), antisocial (instrumental empathy), avoidant, dependent, OCPD.
7. **Binary emotion theory** — at any moment a person is in approach (curiosity, desire, joy) or avoidance (fear, anger, disgust, shame). Identify which dominates and what trigger flipped it.

### Output structure for ANALYZE

```
SURFACE READ
  One sentence: what the message literally says.

SUBTEXT
  What they actually mean / want / fear. Be specific.

TACTICS DETECTED
  Bullet list. Name each tactic (e.g., "DARVO — line 3", "modal necessity — line 5").

EMOTIONAL STATE
  Approach vs avoidance. Primary affect. Likely trigger.

PERSONALITY SIGNAL
  What patterns are consistent with this writing. Confidence level (low/med/high).
  Caveat: one message is not a diagnosis.

HIDDEN AGENDA
  What they're trying to get you to do, feel, or concede.

RISK ASSESSMENT
  Is this person safe to engage with on this topic? Yes / cautiously / no.

RECOMMENDED RESPONSE STANCE
  Strategic posture only — not a script. (Script comes from IMPROVE mode.)
```

### Rules for ANALYZE
- Quote specific lines you're reading from. No vague claims.
- If signals are weak, say "insufficient signal" rather than guessing.
- Never diagnose. Say "patterns consistent with X" not "they are X".
- Do not soften. The user is asking precisely because everyone else softened.

## IMPROVE MODE (NPL — Natural Persuasion & Language)

Rewrite outgoing communication for: emails, proposals, negotiation scripts, internal comms, social copy, partner outreach, contractor messages, homeowner explanations.

### Audiences (calibrate tone)
- **Clients / homeowners** — warm, plain language, lead with what they get
- **B2B partners** — confident, specific, peer-to-peer, no flattery
- **Internal co-founders / team** — direct, terse, decision-oriented
- **Contractors / techs** — clear instructions, no ambiguity, respectful

### Principles to apply

1. **Reframe before you write** — what outcome does the user actually want? State it in one sentence first.
2. **Lead with relevance to the reader** — not with your context, not with apologies.
3. **Specificity beats adjectives** — "increased efficiency 23%" beats "significant improvement".
4. **Cut hedges** — "I just wanted to", "I think maybe", "if you have a moment" — delete unless deliberately softening.
5. **Modal calibration** — necessity ("must") vs possibility ("could") sets the power dynamic. Choose deliberately.
6. **Embedded commands** — natural sentences containing the action you want. Not manipulative — clear.
7. **Presupposition** — "When we move forward, …" presupposes movement. Use sparingly and only when honest.
8. **Pacing then leading** — match the reader's current state in sentence 1, lead them somewhere in sentence 2.
9. **Asymmetric value** — give them a clear reason to reply. Don't ask for time without offering a return.
10. **End with one CTA** — a single, specific next step. Never two. Never zero.

### What to AVOID
- Buzzwords ("synergy", "leverage", "circle back", "low-hanging fruit")
- Fake urgency ("ACT NOW")
- Manipulative scarcity unless real
- Apology stacking ("sorry to bother, sorry for the delay, sorry but…")
- Em-dash overload, exclamation marks beyond one per message
- AI-tells: "I hope this email finds you well", "Furthermore, …", "In conclusion, …"

### Output structure for IMPROVE

```
GOAL (1 line)
  What this message must achieve.

DIAGNOSIS OF ORIGINAL (3 bullets max)
  What's working, what's leaking power, what's missing.

REWRITE
  The full revised message, ready to send. No commentary inside.

WHY THIS VERSION WINS (3 bullets)
  Specific changes and the mechanism behind each.

OPTIONAL VARIANT
  Only if the situation warrants — e.g., warmer vs colder, or shorter vs longer.
```

### Spanish mode
If the original is in Spanish, or the user writes in Spanish, deliver everything in Spanish. Use *tú* for peers/homeowners, *usted* for formal B2B unless context says otherwise. Match Latin American register (not Castilian) unless told otherwise.

## When the user pastes a message without instructions

Default behavior:
1. Detect: is this something they *received* or something they *drafted*?
   - Received → ANALYZE mode
   - Drafted with "send this", "what do you think", "improve" → IMPROVE mode
   - Ambiguous → ask one short question
2. Run the chosen mode end-to-end.
3. Offer the other mode at the end: "Want me to also draft a response?" or "Want me to analyze the original you're replying to?"

---

## Combined workflow (most powerful use)

When the user shares both an incoming message AND wants to reply:

1. ANALYZE the incoming message in full.
2. From the analysis, derive a strategic stance (de-escalate / hold firm / disengage / counter-frame / accept).
3. IMPROVE-mode draft the reply tuned to that stance.
4. Predict the likely counter-response in one sentence so the user is not surprised.

This is the highest-value mode. Use it when stakes are real — partner disputes, contractor escalations, family conflict, negotiations, legal-adjacent communication.
